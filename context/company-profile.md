# FlowBoard - Company Profile

## About
FlowBoard is a cloud-based project management and team collaboration SaaS platform designed for small-to-medium businesses (SMBs) and startup teams. Founded in 2021, FlowBoard helps teams organize tasks, track progress, and collaborate in real-time.

## Product
- **FlowBoard Free**: Up to 5 users, 3 projects, basic task management
- **FlowBoard Pro**: $12/user/month - Unlimited projects, time tracking, integrations, file storage (10GB/user)
- **FlowBoard Business**: $25/user/month - Advanced analytics, custom workflows, priority support, SSO, 50GB/user storage
- **FlowBoard Enterprise**: Custom pricing - Dedicated infrastructure, SLA, dedicated account manager

## Key Features
1. **Task Boards** - Kanban, List, Calendar, and Timeline views
2. **Real-time Collaboration** - Comments, @mentions, live editing
3. **Time Tracking** - Built-in timer, manual entry, reports
4. **Integrations** - Slack, GitHub, Google Drive, Dropbox, Zapier
5. **Automations** - Rule-based task assignment, status changes, notifications
6. **Reports & Dashboards** - Team velocity, burndown charts, workload view
7. **File Sharing** - Attach files to tasks, preview in-app
8. **Mobile Apps** - iOS and Android
9. **API Access** - REST API for Pro and above
10. **Custom Fields** - Text, number, date, dropdown, checkbox
